Events

  • onClick - occurs when the user clicks the left mouse button on an event
  • onEventAdded - occurs when the user adds a new event
  • onEventChanged - occurs after editing an event and saving changes (after clicking on edit and save buttons in event bar or in details window)
  • onBeforeEventDelete - blockable, occurs after user clicks on delete button (in event bar or in details window)
