addEvent(start_date, end_date, text, id, data)

Required library edition:
Required library file: dhtmlxscheduler.js

adds a new event

  • start date - date object or string;
  • end date - date object or string;
  • text;
  • id - optional, will be auto-generated if skipped;
  • other data - hash.
   scheduler.addEvent("16-03-1984","19-03-1984","Some");
//or
   scheduler.addEvent("16-03-1984","19-03-1984","Some","some id",{custom_data:"some data"});
//or
   scheduler.addEvent({
          start_date: "16-03-1984",
          end_date: "19-03-1984",
          text:"Some",
          custom_data:"some data"
   });

scheduler.config.api_date used to parse incoming dates
