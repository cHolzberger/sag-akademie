dhtmlxScheduler

dhtmlxScheduler is user-friendly event scheduler component which allows you to program and schedule any event such as board meeting, business lunch, conference call, family reunion, movie night, or any other group event. You are allowed to create, edit, delete and drag-and-drop events in scheduler.

First of all you should know that dhtmlxScheduler is able to schedule events for days, weeks and months. To set your time mode you should select one of three available modes in the time mode panel located on the right side of calendar panel of the component.

Backward/forward arrows on the left side of calendar panel allow to move within calendar dates. Today button enables you to turn to current date. The selected date period is specified behind today button. Hours scale is located on the left side of the scheduler.

Events Manipulation

Creating a New Event

There are two ways of creating an event in dhtmlxScheduler.

  1. Double click on desired field of date.
  2. Press the right button of the mouse and drag date area to define the date period of the event.

After that an event bar appears, this item of dhtmlxScheduler consists of two bars. Now we are going to discover the functionality of both of them. The left vertical bar contains two buttons: the first one indicated by green tick is aimed to save an event; the second one indicated by red cross allows to delete a created event.

Note:

  • After saving an event new Details and Delete buttons will be added to the bar.

  • The left bar appears only during active state of the event (editing before saving). In normal state event bar consists just of the right one.

The right horizontal bar contains general information about newly created event: time of execution and name of the event.

Editing Event

To specify details about an event you should click on Details button in the left bar of the event. Also, Details window can be called by double click on right bar of the event. Details Window consists of three main fields:

  • description;
  • event type;
  • time period.

Description and Time period fields are initially included in the Details window, but they can be deleted at will. As for the third field, we are using Event type name for it just for example. You are allowed to set any additional fields with custom data. At the foot of the details window there are save, cancel and delete buttons.

Setting a Multidays Event

There is a possibility to set a multidays event. To do this you should switch month mode, then press right button of mouse and drag the desired area of the event.

Drag-and-Drop in dhtmlxScheduler

dhtmlxScheduler supports drag-and-drop, thus you are allowed to move and resize a created event within calendar area accordingly to your needs.

Deleting an Event

There are two ways to delete an event:

  1. Click on appropriate button in the left bar of event.
  2. Call details window and click on delete button inside it.
