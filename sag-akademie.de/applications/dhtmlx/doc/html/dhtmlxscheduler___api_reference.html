dhtmlxScheduler API Methods

  • addEvent - adds new event
  • changeEventId - changes event id
  • clearAll - clears all data in scheduler.
  • deleteEvent - deletes existing event
  • editStop - switches from editing mode to normal mode, disable editor if it's open
  • edit - edits event
  • getEventEndDate - returns end date of event
  • getEventStartDate - returns start day of event
  • getEventText - returns event text
  • getEvent - returns event object by id
  • load - load data in scheduler
  • select - selects event
  • setCurrentView - sets the current mode and displayed date
  • setEventEndDate - sets end date of event
  • setEventStartDate - sets start date of event
  • setEventText - sets event text
  • setLoadMode - sets auto-loading mode
  • showLightbox - shows details form for the event:
  • uid - returns unique ID ( unique inside current scheduler, not GUID )
  • unselect - unselects event
  • updateEvent - forces event updating
