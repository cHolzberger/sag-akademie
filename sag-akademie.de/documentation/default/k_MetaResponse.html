k_MetaResponse

Description
Description | Descendents | Vars | Methods (details)

A metaresponse represents an abstract event in the application, which needs alternate handling.

This would typically be an error-condition. In the simplest invocation, a metaresponse maps directly to a component, which renders a generic error.
