strptime (line 19)

Parse a time/date generated with strftime().

This function is the same as the original one defined by PHP (Linux/Unix only), but now you can use it on Windows too. Limitation : Only this format can be parsed %S, %M, %H, %d, %m, %Y

  • return: Returns an array with the
    1. $sDate
    parsed, or
    1. false
    on error.

  • version: 1.0

(array) strptime ($sDate(string) $sDate, $sFormat(string) $sFormat)
  • $sDate(string) $sDate: The string to parse (e.g. returned from strftime()).
  • $sFormat(string) $sFormat: The format used in date (e.g. the same as used in strftime()).
