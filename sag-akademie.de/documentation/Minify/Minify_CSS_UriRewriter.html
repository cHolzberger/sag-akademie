				$symlinks: (default = array()) If the CSS file is stored in  a symlink-ed directory, provide an array of link paths to  target paths, where the link paths are within the document root. Because  paths need to be normalized for this to work, use "//" to substitute  the doc root in the link paths (the array keys). E.g.:  
    1.  array('//symlink' => '/real/target/path') // unix
      2. 

    2.  array('//static' => 'D:\\staticStorage')  // Windows
